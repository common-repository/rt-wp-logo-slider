Logo Slider plugin is fully Responsive and Mobile Friendly WordPress Logo Slider plugin to display your clients or partners logo in slider and grid views.

[youtube https://www.youtube.com/watch?v=CRM34AlqO4Y]

You can easily manage Logo from admin end and can create unlimited shortcode. You can arrange logo Slider by category wise.

From settings you can set Logo size and can set custom detail URL link for each Logo.

You can display Logo Slider by shortcode into page or post or you can use widget to display in widget positions.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'WP Logo slider'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `rt-wp-logo-slider.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `rt-wp-logo-slider.zip`
2. Extract the `rt-wp-logo-slider` directory to your computer
3. Upload the `rt-wp-logo-slider` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard

= How to Use Post Grid=

* Go to `Logo slider > Add New Logo`
* Go to `Logo slider > Shortcode Generator` create new shortcode
* Go to page or post editor click Logo Showcase Shortcode button and select a shortcode from dropdown list and insert shortcode.
